# Coffee-limiter

## Project Description

This project is a coffee-limiter that uses a PN532 NFC reader to detect a card and look up the user on Hackerspace's website to see if they are allowed more coffee today.

## Project Development

### On VSCode

1. [Install gcc](https://github.com/niXman/mingw-builds-binaries/releases) and add it to path so you may compile the project.
2. [Install PlatformIO IDE.](https://docs.platformio.org/en/latest/integration/ide/vscode.html) This is a useful IDE for VSCode that simplifies running and debugging on the Arduino Uno.
